# Contributing

Thanks for taking the time to look at SHA.

This repository is published primarily as a working public codebase, not as a high-volume community project. Contributions are welcome, with review and merge decisions handled on a best-effort basis.

## Before You Start

- Open an issue before larger changes so direction stays aligned.
- Keep pull requests small, focused, and easy to review.
- Preserve existing architecture and coding patterns unless the change intentionally restructures them.
- Add or update tests when behavior changes.

## Development

Install, run, and validation commands are documented in [README.md](README.md).
